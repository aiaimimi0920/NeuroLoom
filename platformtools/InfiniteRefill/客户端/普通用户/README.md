# 无限续杯客户端使用说明（Windows / macOS / Ubuntu）

## 一、快速开始

1. 打开配置文件：`无限续杯配置.env`
2. 填写或后续在菜单中填写：
   - `USER_KEY`
3. 按系统运行入口脚本：
   - Windows：`windows/无限续杯.bat`
   - macOS / Ubuntu：`unix/无限续杯.sh`

## 二、菜单推荐流程（两端一致）

1. 选择 `2`：设置/更新配置（服务器地址、用户密钥、账号目录、同步目录、间隔）
2. 选择 `3`：开启/更新定时续杯
3. 需要立即补齐时选择 `1`：立即执行一次单次续杯
4. 需要全量回收最新账号时选择 `5`：同步所有账号

## 三、常见问题

- 出现异常状态或账号不一致，优先执行一次“同步所有账号”、“再尝试删除accounts目录下内容后同步所有账号”、最后@我。
- 若发现定时任务未生效，检查：
  - `crontab -l` 是否存在对应任务块
  - `unix/_定时任务_入口.sh` 是否可执行
- 若服务端返回自动禁用/封禁，客户端会自动停用定时任务，需先处理服务端侧限制再重新启用。（说明你可能调用过于频繁、或者服务端认为你滥用，还是@我）

## 四、安全提醒

- `无限续杯配置.env` 含密钥，禁止外传。
- `accounts/*.json` 也属于敏感凭据文件，禁止上传公共仓库。
